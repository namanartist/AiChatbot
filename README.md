# NamanAI - Advanced Rule-Based AI Chatbot

Welcome to **NamanAI**, a complete, professional, and internship-ready rule-based Artificial Intelligence chatbot application written in Python 3. NamanAI demonstrates core AI fundamentals including input preprocessing, intent recognition, control flow decision-making, exception handling, dynamic content arrays, statistics tracking, and persistent history logging.

This project is built from scratch without any external dependencies, ensuring maximum compatibility across platforms (including Windows).

---

## Features

- **Object-Oriented Design**: Encapsulated within the structured `NamanAI` class.
- **Robust Intent Recognition**: Clean dictionary-based mapping matching user intents instantly.
- **Safe Calculator**: Implements a math expression evaluator using Python's built-in Abstract Syntax Tree (`ast`) module. It avoids unsafe `eval()` executions and handles invalid math structures or division-by-zero errors gracefully.
- **Keyword-Based Mood Detection**: Scans user inputs to detect emotional keywords like "sad", "happy", or "tired" and returns helpful motivational messages.
- **Continuous Conversation Loop**: A robust command-line loop that handles empty inputs and keeps running without crashing.
- **Session Statistics**: Real-time tracking of total messages processed, recognized commands, and unknown/fallback commands.
- **Persistent History Log**: Memory-buffered conversation turns are automatically saved to `chat_history.txt` in a clean timestamped format upon exiting.
- **Zero External Dependencies**: Runs entirely on the Python Standard Library.
- **Platform Compatible & Clean Encoding**: Fully tested on Windows, ensuring no encoding bugs or Unicode emoji rendering issues.

## Getting Started

### Prerequisites
- Python 3.12 or newer installed on your machine.

### Running the Chatbot
1. Open your terminal or Windows PowerShell.
2. Navigate to the project folder:
   ```bash
   cd NamanAI
   ```
3. Run the script:
   ```bash
   python naman_ai.py
   ```

## Supported Commands

| Command Category | Command Syntax | Description |
|---|---|---|
| **Greetings** | `hello`, `hi`, `hey` | Returns a random friendly greeting variation. |
| | `good morning`, `good afternoon`, `good evening`, `good night` | Tailored responses based on the time of day. |
| **About** | `who are you` | Description of the chatbot. |
| | `what is your name` | Returns the chatbot name. |
| | `creator`, `developer` | Displays information about Naman Lahariya. |
| | `version` | Displays the current version details. |
| **Technology** | `ai`, `machine learning` | Defines core AI and ML concepts. |
| | `python`, `java`, `react`, `mongodb`, `mern` | Explains key languages, frameworks, and databases. |
| **Career** | `motivate me` | Returns a motivational phrase along with a quote. |
| | `goal`, `future`, `coding` | Actionable career guidance and coding tips. |
| **Profile** | `github`, `linkedin`, `portfolio` | Provides links to Naman Lahariya's professional web pages. |
| | `projects` | Lists notable projects developed by the creator. |
| **Utility** | `date` | Returns the current system date. |
| | `time` | Returns the current system time. |
| **Fun** | `joke` | Selects a random programming joke. |
| | `quote` | Selects a random motivational quote. |
| | `fact` | Selects a random AI fact. |
| | `tip` | Selects a random software engineering tip. |
| **Tools** | `flip coin` | Simulates flipping a coin (Heads/Tails). |
| | `roll dice` | Simulates rolling a six-sided dice (1-6). |
| **Calculator** | `calculate <expression>` | Safely evaluates math expressions (e.g. `calculate 5+5`, `calculate 100/2`). |
